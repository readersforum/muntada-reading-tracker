# سِجِلّ القراءة — منتدى النص والقارئ

تطبيق تلكرام مصغر (Telegram Mini App) لتسجيل ومتابعة القراءة اليومية لأعضاء المنتدى.

## التشغيل محلياً

```bash
npm install
npm run dev
```

يفتح على `http://localhost:5173`. ملاحظة: خارج تلكرام، سيستخدم التطبيق `localStorage` بدلاً من CloudStorage، وحقل الاسم ما رح يتعبى تلقائياً.

## البناء للنشر

```bash
npm run build
```

يطلع مجلد `dist/` جاهز للاستضافة.

## النشر على Vercel (الأسهل)

1. ارفع هذا المجلد إلى مستودع GitHub.
2. روح إلى [vercel.com](https://vercel.com) وسجّل دخول بحساب GitHub.
3. اختر "New Project" واربطه بالمستودع.
4. Vercel بيكتشف تلقائياً إنه مشروع Vite، اضغط Deploy.
5. بعد انتهاء البناء رح تحصل على رابط HTTPS مثل `https://muntada-reading.vercel.app`.

(نفس الخطوات تنطبق على Netlify أو GitHub Pages إذا تفضلها.)

## ربطه ببوت تلكرام

1. كلّم [@BotFather](https://t.me/BotFather) على تلكرام.
2. إذا ما عندك بوت: أرسل `/newbot` واتبع التعليمات (اسم + username ينتهي بـ bot).
3. أرسل `/newapp`، اختر البوت، واملأ:
   - اسم التطبيق (مثلاً "سجل القراءة")
   - وصف مختصر
   - صورة/أيقونة (تقدر تستخدم شعار المنتدى نفسه)
   - **رابط الويب اب**: الصق رابط Vercel اللي حصلت عليه.
4. لإضافته كزر دائم بالبوت: أرسل `/setmenubutton`، اختر البوت، وحط نفس الرابط مع نص الزر (مثلاً "📖 سجل قراءتك").
5. جرّب فتح البوت من تلكرام واضغط الزر — لازم يفتح التطبيق ويتعبى اسمك تلقائياً من حساب تلكرام.

## ربطه بقناة المنتدى (@muntada25)

تقدر تنشر رسالة بالقناة فيها زر Inline يفتح التطبيق مباشرة (عبر الـ Bot API `sendMessage` مع `inline_keyboard` من نوع `web_app`)، أو ببساطة توجّه الأعضاء لفتح البوت من الوصف.

## ملاحظة مهمة: لوحة القرّاء المشتركة

خيار "أظهر تقدمي في لوحة القرّاء المشتركة" موجود بالواجهة لكنه حالياً يخزن فقط تفضيل المستخدم محلياً. تلكرام CloudStorage يخزّن بيانات كل مستخدم بشكل خاص فقط — ما يقدر مستخدم يشوف بيانات مستخدم ثاني. لعرض لوحة صدارة حقيقية بين الأعضاء تحتاج قاعدة بيانات بسيطة (مثل Supabase أو Cloudflare Workers + KV) يكتب لها كل عضو ويقرأ منها الجميع. إذا حاب أجهزلك هذا الجزء لاحقاً، خبرني.
